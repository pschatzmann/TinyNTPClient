Detailed Description

template<typename UDPAPI>
class TinyNTPClient< UDPAPI >

A Network Time Protocol (NTP) client for retrieving the current date/time from an NTP server using an Arduino UDP API. It returns the time information from the NTP server and adjusts it based on the round-trip delay calculation. The timezone offset can be set to account for local time differences.

Template Parameters
UDPAPIThe UDP class to use (e.g: WiFiUDP)
